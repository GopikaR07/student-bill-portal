# Student Bill Portal

A Flask-based web application for students to submit bill reimbursement requests
and for admins to verify, approve, or reject them.

## Features
- Student & Admin login
- Multiple product bill submission
- PDF upload (Bill & Passbook)
- Request status tracking
- Admin approval / rejection
- Excel export of approved requests

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript
- Git & GitHub

## How to Run
1. Install Python 3
2. Install dependencies:
   pip install flask pandas
3. Run the app:
   python app.py
4. Open:
   http://127.0.0.1:5000
